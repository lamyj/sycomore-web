Interactive MRI simulations

This applications presents interactive simulations of common MRI experiments using Sycomore for the simulation itself and Bokeh for the graphical user interface.

  • RARE: acquisition of multiple lines per TR (RARE imaging: A fast imaging method for clinical MR, Hennig et al., 1986, aka Fast Spin Echo or Turbo Spin Echo), using Bloch simulation
  • RF spoiling (Spoiling of transverse magnetization in steady-state sequences, Zur et al., 1991), using regular EPG
    • Evolution of an RF- and gradient-spoiled signal with respect to phase increment
    • Efficiency of the RF-spoiling for all phase steps
  • Contrasts of a spin-echo sequence
  • Slice profile: transversal and longitudinal signal magnitude when using a selective sinc pulse, using 3D discrete EPG
